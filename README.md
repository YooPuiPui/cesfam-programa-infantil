# Sistema de Gestión Clínica - CESFAM Programa Infantil

Arquitectura inicial en contenedores (Frontend, Backend y Base de Datos) para el sistema del Programa Infantil del CESFAM Santa Sabina.

## Instrucciones de Despliegue 

Para levantar el proyecto completo de forma orquestada, ejecute exactamente los siguientes comandos en su terminal:

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/tu-usuario-de-github/cesfam-programa-infantil.git](https://github.com/tu-usuario-de-github/cesfam-programa-infantil.git)

2. **Ingrear al directorio:**
    cd cesfam-programa-infantil

3. **Levantar el proyecto**
    docker compose up
